# Common Voice based language recognition deep learning project
This repository is for a university deep learning course final project - Common Voice based language recognition.

## Dataset: [Common Voice](https://mozilladatacollective.com/organization/cmfh0j9o10006ns07jq45h7xk)

We chose 4 languages (Polish, German, Portuguese, Esperanto - if you consider it a language) from different language families to minimize issues from the very start. Originally, plan was for 3 languages, but since Common Voice datasets for German and Esperanto were unbelievably small compared to Polish (also, we weren't sure whether the professors were going to allow Esperanto since it's, well, Esperanto), we decided to include Portuguese in there as well. Prediciton: the results for Polish and Portuguese will be the most loyal and have the smallest rate of false-positive. We'll see if it's true ;-)
